# Pokevite

## Como rodar projeto?

Basta abrir seu terminal e rodar os seguintes comandos:

```sh
git clone https://github.com/tamiramiranda/Pokevite.git
```

```sh
cd PokeVite
```

```sh
npm i
```

```sh
npm run dev


npm run build
```

Após isso provavelmente o projeto estará rodando em http://localhost:5173/
